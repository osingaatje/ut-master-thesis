# ut-master-thesis
Master's thesis for the University of Twente (2025-2026)

**NOTE**: This repository is mirrored from [Codeberg](https://codeberg.org/drwr/ut-master-thesis) to [GitHub](https://github.com/osingaatje/ut-master-thesis).

## Compiling the thesis
For generating the PDF, please make sure to install the fonts in the [`typst-template-ut`](https://codeberg.org/drwr/typst-template-ut) project, follow the README in that project, and then run:

```sh
typst compile fp.typ   # ...or for the Research Topics: rt.typ
```
